# Классификатор изображений
Приложение позволяет создать бинарный классификатор изображений, используя небольшое количесто данных, которые можно набрать через встроенный веб-поиск.

## Архитектура
**Основной стэк:** Python, FastAPI, PyTorch, torchvision, scikit-learn, matplotlib, TypeScript, React, Docker

Для дообучения (fine-tuning) используется модель **ResNet18** с предобученными весами на базе изображений **ImageNet**. Дообучение с использованием предобученных весов позволяет получить более качественную модель, благодаря переносу знаний модели (**transfer learning**). Модель демонстрирует хорошие показатели скорости и качества обучения на небольших датасетах и пользовательских процессорах (~1 минута на датасете из 50 изображений), что дает хороший пользовательский опыт при взаимодействии через UI.

Обучение происходит методом стохастического градентного спуска (**SGD**). Для оптимизации обучения используется алгоритм **AdamW** (эффективный подбор гиперпараметров в ходе обучения), а также механизм  ранней остановки обучения с выбором лучших весов. Данные деляться на три части: **train** (на них происходит тренировка), **val** (валидация успешности модели во время обучения) и **test** (финальная проверка работы модели). Для улучшения качества генерализации применяется аугментация тренировачных данных (**Data Augmentation**): наклон изображения, отражение по горизонтали, изменение цвета.

## Установка и команды
```bash
git clone https://github.com/7ladde7/repo/ml-finetune-image-classifier
cd ml-finetune-image-classifier
```

Запуск в **Docker**:
```bash
docker compose up --build

# метрики
python -m train.evals
```
Открыть в браузере: [http://localhost:3000](http://localhost:3000)

## Метрики
Ключевые результаты:
| модель                         | samples | tr_loss | tr_acc | val_loss | val_acc | recall | precision | f1   |
|------------------------------|--------:|--------:|--------:|----------:|--------:|--------:|----------:|------:|
| CNN                          | 50      | 0.67    | 0.57    | 0.67      | 0.40    | 0.82    | 0.50      | 0.62 |
| ResNet18_scratch             | 50      | 0.38    | 0.85    | 0.53      | 0.70    | 0.76    | 0.56      | 0.64 |
| ResNet18_finetuned           | 50      | 0.09    | 0.97    | 0.06      | 1.00    | 0.97    | 0.91.     | 0.94 |
| CNN                          | 250     | 0.55    | 0.71    | 0.65      | 0.68    | 0.78    | 0.56      | 0.65 |
| ResNet18_scratch             | 250     | 0.62    | 0.65    | 0.61      | 0.66    | 0.82    | 0.55      | 0.66 |
| ResNet18_finetuned           | 250     | 0.03    | 0.98    | 0.01      | 1.00    | 0.98    | 0.98      | 0.98 |
| CNN                          | 1000    | 0.57    | 0.73    | 0.62      | 0.64    | 0.70    | 0.67      | 0.68 |
| ResNet18_scratch             | 1000    | 0.51    | 0.75    | 0.62      | 0.67    | 0.74    | 0.66      | 0.70 |
| ResNet18_finetuned           | 1000    | 0.07    | 0.97    | 0.03      | 0.98    | 0.98    | 0.97      | 0.97 |

- **CNN** - классический алгоритм сверточной нейронной сети со случайно инициализированными весами.
- **ResNet18_scratch** - нейронная сеть **ResNet18**, обучение с нуля.
- **ResNet18_finetuned** - fine-tuning **ResNet18** на предобученных весах (**transfer learning**).
